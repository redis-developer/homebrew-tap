# Homebrew Tap for Redis Developer Tools

This is a Homebrew tap for Redis developer tools.

## Installation

```bash
brew tap redis-developer/tap
```

## Available Formulas

### redisctl

CLI for Redis Cloud and Enterprise management.

```bash
brew install redis-developer/tap/redisctl
```
